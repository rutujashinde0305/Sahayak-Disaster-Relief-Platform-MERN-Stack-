# Sahayak — Disaster Relief Platform

This repository contains the Sahayak Disaster Relief project (frontend + backend).

- Frontend (Vite + React): `my-project/` — see `my-project/README.md` for detailed setup and screenshots.
- Backend (Express + MongoDB): `backend/` — see `backend/` for environment and start instructions.

Quick links:
- `my-project/README.md` — frontend quickstart & screenshots
- `backend/` — backend server and env examples

To run locally (frontend):

```powershell
cd my-project
npm install
npm run dev
```

To run the backend (in a separate terminal):

```powershell
cd backend
npm install
# configure .env (MONGO_URI, PORT=3001, etc.)
npm start
```
